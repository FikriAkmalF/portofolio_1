SKILL

Di zaman modern saat ini banyak perusahaan yang membutuhkan tenaga kerja IT yang dapat membangun website serta menciptakan antarmuka pengguna yang menarik dan responsif. Maka dari itu di zaman sekarang harus memiliki skill yang dibutuhkan untuk menjadi Front End Developer. Berikut ini skill yang dimiliki saya :

CSS

Coascading Style Sheet atau CSS adalah aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. Bisa dirangkum CSS adalah bahasa pemograman untuk mengatur posisi agar terlihat rapih dan berwarna.

HTML

Hypertext Markup Language atau HTML adalah bahasa pemograman dasar yang harus di kuasai oleh orang yang ingin menjadi programer, bahasa ini dapat dikombinasikan bahasa lain seperti : CSS & JAVASCRIPT

JavaScript

JavaScript adalah suatu bahasa pemrograman tingkat tinggi dan dinamis. JavaScript populer di internet dan dapat bekerja di sebagian besar penjelajah web populer seperti Google Chrome, Internet Explorer, Mozilla Firefox, Netscape dan Opera.
